Программный продукт «СОФТКОМ: Подсистема «Резерв отпусков» является оригинальной разработкой Компании ООО «СОФТКОМ групп». Продукт выступает как дополнение к конфигурациям «1С:Предприятие 8».

Назначение подсистемы «Резерв отпусков» для «1С:УТП»

:

  • расчет суммы резерва отпусков;
  • расчета ЕСВ Фот на сумму начисленного;
  • автоматизированное  формирование проводок по начисленной сумме резерва и ЕСВ Фот на эту сумму в бухгалтерском  учете;
  • ведение учета отпусков в разрезе их видов.

Назначение подсистемы «Резерв отпусков для «1С:УПП» :

  • расчет суммы резерва отпусков;
  • расчета ЕСВ Фот на сумму начисленного;
  • автоматизированное  формирование проводок по начисленной сумме резерва и ЕСВ Фот на эту сумму в бухгалтерском  учете;
  • получение отчетности по количеству дней и сумме выплаченных отпускных за период;
  • корректировка количества положенных дней отпуска по сотрудникам находящимся в отпуске по уходу за ребенком без оплаты.

Назначение подсистемы «Резерв отпусков для «1С:ЗУП» :

  • расчет суммы резерва отпусков;
  • расчета ЕСВ Фот на сумму начисленного;

Кому данные решения будет полезными:

  • Отдел по расчету заработной платы;
  • Отдел кадров.
